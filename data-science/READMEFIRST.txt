***************************************************************************INSTRUCTIONS****************************************************************************

**********************************************************************PREFERRED OPTION ****************************************************************************
To set up the environment, please make sure you have docker installed.

sh runDocker.sh 

This will create an image and run jupyter notebook mounting the current directory.


visit http://localhost:<port_number>/tree? (usually port number will be 32768 and above) to see the notebook 

docker ps 

To know whats the port in which the service is runing.


Either see the Notebook file or directly run sh start.sh to see the results in the results folder in csv format

*********************************************************************ALTERNATE OPTION*******************************************************************************

Run jupyter notebook from the directory but please make sure you install the requirements first making use of the requirements.txt file. 

Either see the Notebook file or directly run sh start.sh to see the results in the results folder in csv format
********************************************************************************************************************************************************************
